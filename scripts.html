Big Sale

Everything in the store has been discounted by 20%. You are given a program that takes the price of an item as input. Complete the program so that it outputs discounted price.

Sample Input

100, 150

Sample Output

80, 120

Explanation

20 percent of 100 equals to 20 (100*20/100), so the discounted price will be 80 (100-20).

Remember the division (/) and multiplication (*) operators.

Output: newPrice

We use these operators to assign values to Javascript variables. We can use multiple assignment operators in one line, such as x -= y+= 9.

Operator Example Is equivalent to

-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y

Comparison Operators

We can use comparison operators in logical statement to find outif variables or values are different.

You can get either true or false. For example, the equal to (==) operator checks whether the operands' values are equal.

You can compare all types of data with comparison operators, they'll always return true or false.

Operator Description Is equivalent to
== Equal to 5==10 false
=== value&type 5===10 false
!= Not equal to 5!=10 true
!== value&type 10!==10 false
> Greater than 10>5 true
>= Greater than or equal to 10>=5 true
< Less than 10<5 false
<= Less than or equal to 10<=5 false

Logical Operators

Logical Operators, also known as Boolean Operators, evaluate an expression and return true or false.

Check out the table below to see more details on the logical operators (AND, OR, NOT)

Logical Operators
&& Returns true, if both operands are true
∣∣ Returns true, if one operands are true
! Returns true, if the operands is false, and false, if the operand is true
